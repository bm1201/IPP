## 스프링부트를 활용한 게시판
### 1. 프로젝트 설명
대전천변도시고속화도로 프로젝트에 게시판기능 추가

### 2. 작업내용
- 홈페이지 메뉴
  - 로그인을 하지 않아도 게시판메뉴 접속가능(7/26추가)
- 게시글 CRUD(7/16추가)
   - 게시글 검색 및 페이징(7/19추가)
   - 게시글 등록자가 로그인한 ID의 사용자 이름으로 고정되게 수정(7/21추가)
   - 로그인을 한 회원만 게시글을 등록할 수 있도록 수정(7/22추가)
- 게시글 비밀 기능()
- 화면 레이아웃 기존 프로젝트 소스에 맞춰서 수정(7/28추가)
### 3. 사용기술
- Thymeleaf
- Restful API
- SpringBoot
